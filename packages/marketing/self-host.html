Self-host Careerpin

Run Careerpin on your own machine. Your data stays local. No account required.

What You Get

  • Full Careerpin product (same code as the hosted version)
  • Your data stays on your machine (PostgreSQL database, local file storage)
  • No telemetry, no phone-home, no tracking
  • AI features available if you provide your own Anthropic API key
  • Free forever

Prerequisites

  • Docker — Install from docker.com
  • Docker Compose — Usually included with Docker Desktop
  • Terminal access — You'll run a few commands
  • (Optional) Anthropic API key — For AI features. Get one from console.anthropic.com

Quick Start

Clone the repository and start the services:

git clone https://github.com/wickidcool/job-app-management.git
cd job-app-management
cp packages/api/.env.example packages/api/.env

Edit packages/api/.env and set your configuration:

# Required
DATABASE_URL=postgresql://careerpin:careerpin@db:5432/careerpin

# Optional: Enable AI features
ANTHROPIC_API_KEY=your-api-key-here

# Optional: Customize ports
PORT=3000
HOST=0.0.0.0

Start Careerpin:

docker compose up -d

Run database migrations:

docker compose exec api npm run db:migrate

Open your browser to http://localhost:5173

Architecture

Self-hosted Careerpin runs three services:

  • PostgreSQL database — Stores your career catalog, applications, artifacts
  • API server — Node.js/Fastify backend (port 3000)
  • Frontend — React SPA served by Vite (port 5173)

All data is stored locally in Docker volumes. To back up your data, export the PostgreSQL volume or use Careerpin's built-in export feature.

AI Features

To enable AI-powered features (resume variants, job-fit analysis), you need an Anthropic API key:

  1. Sign up at console.anthropic.com
  2. Create an API key
  3. Add it to packages/api/.env as ANTHROPIC_API_KEY=...
  4. Restart the API service: docker compose restart api

Usage costs: You'll be billed by Anthropic at their standard rates. Careerpin uses the claude-sonnet-4-6 model by default. Typical resume generation costs ~$0.02–0.05 per variant.

Updates

To update your self-hosted instance to the latest version:

cd job-app-management
git pull origin main
docker compose down
docker compose build
docker compose up -d
docker compose exec api npm run db:migrate

Backup & Data Export

Two options:

1. Built-in export (easiest)

Use Careerpin's export feature in Settings. Download your catalog as JSON and your resumes as PDF.

2. Database backup (complete)

docker compose exec db pg_dump -U careerpin careerpin > backup.sql

Restore from backup:

cat backup.sql | docker compose exec -T db psql -U careerpin careerpin

Troubleshooting

Port 5173 is already in use

Another service is using port 5173. Either stop that service or change Careerpin's frontend port in docker-compose.yml.

AI features aren't working

Check that ANTHROPIC_API_KEY is set in packages/api/.env and restart: docker compose restart api

Database migration fails

Ensure the database is running: docker compose ps. If db is unhealthy, check logs: docker compose logs db
